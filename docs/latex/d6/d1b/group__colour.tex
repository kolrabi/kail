changes the contrast of an image by using interpolation and extrapolation. 

Common values for Contrast are in the range -\/0.\-5 to 1.\-7. Anything below 0.\-0 generates a negative of the image with varying contrast. 1.\-0 outputs the original image. 0.\-0 -\/ 1.\-0 lowers the contrast of the image. 1.\-0 -\/ 1.\-7 increases the contrast of the image. This effect is caused by interpolating (or extrapolating) the source image with a totally grey image. \begin{DoxyRefDesc}{Bug}
