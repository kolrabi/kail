Sharpens an image. 

Can actually either sharpen or blur an image, depending on the value of Factor. ilu\+Blur\+Avg and ilu\+Blur\+Gaussian are much faster for blurring, though. When Factor is 1.\+0, the image goes unchanged. When Factor is in the range 0.\+0 -\/ 1.\+0, the current image is blurred. When Factor is in the range 1.\+0 -\/ 2.\+5, the current image is sharpened. To achieve a more pronounced sharpening/blurring effect, simply increase the number of iterations by increasing the value passed in Iter.
