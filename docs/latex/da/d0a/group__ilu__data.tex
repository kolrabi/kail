Compares the current image to the image having the name in Comp. 

If both images are identical, I\-L\-\_\-\-T\-R\-U\-E is returned. I\-L\-\_\-\-F\-A\-L\-S\-E is returned if the images are not identical. The bound image before calling this function remains the bound image after calling il\-Compare\-Image.
